关于1566创客公益基金：

        1566创客公益基金是中国首个创客公益基金，由民间爱心人士通过互联网众筹发起，致力于创客运动发展、创客文化传播、创客环境建设、资助创客成长。
        我们这个世界问题很多，批评者众、行动者少。创客不一样，创客(Maker)或极客(Geek)不仅对世界抱有自己的观点，还能运用各种DIY手段拿出解决方案表达观点。创新也是一种表达，饱含人文精神的科学思想。创客门槛不高，在社会、艺术、文化、技术等任一方面有见解和动手能力均可，小孩子也可当创客。
        如今，中国的创客运动正轰轰烈烈的开展起来，许多投资机构、孵化器、大企业都期待着创客中成长出丰硕成果。但是创客的活动，因其处于萌芽阶段，具有鲜明的非功利性特点，如果大家只是旁观，而不给予实际支持、推动，就很难形成足够的创客人才密度，和丰硕的创意作品。
        1566创客公益基金，对于纯公益项目采取无偿资助，对于潜在的商业化项目，在未商业化阶段不求任何经济回报。该基金的发起运作模式，受到国家、省、市各级部门和机构的赞扬与鼓励，多家重要媒体予以重点关注。爱心公益，不落人后！近朱者赤、与善为邻！
